===== Настройка цветовой схемы в xfce4-terminal =====
# отображение 256 цветов в терминале XFCE
$ echo export TERM='xterm-256color' >> $HOME/.bashrc

Правка->Параметры->Цвета
**Фон**: 58/58/58
**Шрифт**: 201/193/149

===== Настройка mc =====
В меню **Настройки->Оформление** выбрать "Скин: **modarin256**"

Запуск mc с вручную заданной темой:
$ mc –skin=modarin256

===== Свертывание приложений в системный лоток =====
AllTray — приложение для сворачивания приложений в системный трей.

AllTray легко поможет свернуть в системный трей практически любое приложение, разработчики которого не предусмотрели этой функциональной возможности.

После запуска AllTray работает в фоновом режиме, если закрыть окно приложения, то оно не закроется, а просто свернётся в трей. Значки для приложений берутся автоматически из ассоциированных с приложением, закрыть приложения, находящиеся в системном трее, можно из контекстного меню значка (по правому клику мыши).
AllTray работает в любом графическом окружении (Gnome, KDE, XFCE 4, LXDE и пр.), запускать приложения (с alltray) можно отредактировав запуск в меню, а уже запущенные приложения сворачиваются запуском из меню AllTray и кликом появившимся вместо курсора крестиком по нужному приложению.

Запуск:
{{{code: lang="sh" linenumbers="False"
$ alltray -s program_name
}}}
